# README

This is the TU Delft starterkit for open publishing with Jupyter Book. It provides a skeleton with basic settings to have a headstart with your project.

It includes:
- a github deploy file taking care of deploying the website and building a pdf
- a link to the documentation 
- basic content files that can be altered to fit your project
- a link to the Typst template
- yml files in

Push button at landing page for use this template.
